Elizabeth is a Maths teacher. She wants to develop an application to verify the answers to students homework.  Develop an application to do the addition for two complex numbers using Friend classes in C++.

Complex number contains the real numbers and a specific element denoted i, called the imaginary unit. 

Example: a+bi

Operation

 

int addRealPart(Addition obj1, Addition obj2)

 

This method adds the real part of the two complex numbers and return the sum of the real part.

Operation

 

int addImaginaryPart(Addition obj1, Addition obj2)

 

This method adds the imaginary part of the two complex numbers and returns the sum of the imaginary part.

 

In the main method,

  • Get all the inputs from the user as portrayed in the sample input and set the value to the Addition objects using the parameterized constructor.
  • Call the addRealPart() and the addImaginaryPart() method to calculate the sum of the real and the imaginary part of two complex numbers.
  • Display the output as shown in the sample output. 

 Note: Test the methods by invoking them in main method. Only business logic is tested in this question and not presentation. Refer the sample input / output to check the correctness of your code.

Note: Avoid the usage of exit(0) in your code.  

Sample Input:

Enter real and imaginary part of the first complex number

1

2

Enter real and imaginary part of the second complex number

1

2

Sample Output:

Sum of real and imaginary part is 2+i4
